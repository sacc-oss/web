Me parece que el tema de HTML es muy interesante y útil. Aprenderlo me ayuda a entender cómo se construyen las páginas web que usamos todos los días. Aunque al principio parece complicado por tantas etiquetas, poco a poco se vuelve más fácil de usar.
Creo que HTML es una buena base para empezar a programar, porque no se necesita saber mucho para ver resultados. Además, me gusta que con solo unas líneas de código ya puedo crear una página con texto, imágenes y enlaces. En general, es un tema que me gustó y que me sirve para el futuro si quiero aprender más sobre diseño web o programación.
